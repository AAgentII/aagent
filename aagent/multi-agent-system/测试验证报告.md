## API使用示例

### 1. 健康检查
```bash
curl http://localhost:8000/api/health
```

### 2. 创建工作流
```bash
curl -X POST http://localhost:8000/api/workflows \
  -H "Content-Type: application/json" \
  -d '{
    "name": "测试工作流",
    "description": "测试用的工作流",
    "definition": {
      "nodes": [
        {"id": "start", "type": "start", "config": {}},
        {"id": "agent1", "type": "agent", "config": {}, "agent_config": {"name": "worker1", "role": "worker"}},
        {"id": "end", "type": "end", "config": {}}
      ],
      "edges": [
        {"from_node": "start", "to_node": "agent1"},
        {"from_node": "agent1", "to_node": "end"}
      ]
    }
  }'
```

### 3. 执行工作流
```bash
curl -X POST http://localhost:8000/api/executions \
  -H "Content-Type: application/json" \
  -d '{
    "workflow_id": "工作流ID",
    "input_data": {"test": "data"}
  }'
```
